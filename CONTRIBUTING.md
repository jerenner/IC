## Quick Start

1. Fork the IC repository on GitHub
2. Clone your fork and set up the environment:

   ```bash
   git clone git@github.com:<your-username>/IC.git
   cd IC
   source manage.sh work_in_python_version 3.8
   ```

3. Create a topic branch from `upstream/master`
4. Do your work, commit frequently
5. Squash and clean up your commits before submitting
6. Push and open a pull request
7. Wait for review, make changes if requested
8. Once merged, delete your topic branch

For step-by-step magit and git CLI instructions, see the [Contributing Guide](https://next-exp-sw.readthedocs.io/en/latest/IC.html#contributing).
